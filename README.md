# optlib
A Python computational physics library for Monte Carlo simulations of electron-solid interactions, dielectric energy loss functions, and secondary electron yields.

Make sure you have Python [installed properly](https://docs.python-guide.org/).
```
$ git clone git@github.com:oridzel/optlib.git
$ cd optlib
$ pip install -r requirements.txt
$ python setup.py install
```
or
```
pip install git+https://github.com/oridzel/optlib.git
```
